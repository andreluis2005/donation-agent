# Instruções para agentes de IA

Este repositório é mantido para permitir continuidade entre diferentes LLMs e pessoas.

Antes de propor ou editar código, leia nesta ordem:

1. `docs/PROJECT_CONTEXT.md`
2. `docs/DECISIONS.md`
3. `docs/ACTIVE_TASK.md`
4. `docs/AI_HANDOFF.md`

Regras obrigatórias:

- Não sobrescreva alterações não commitadas que não foram feitas por você.
- Atualize `docs/ACTIVE_TASK.md` sempre que iniciar, pausar ou concluir uma tarefa relevante.
- Registre em `docs/DECISIONS.md` decisões que alterem arquitetura, segurança, rede, persistência ou integração externa.
- Não inclua segredos, valores de `.env.local`, chaves privadas ou dados pessoais nos documentos.
- Antes de encerrar uma tarefa de implementação, execute a validação adequada e registre o resultado no handoff.
